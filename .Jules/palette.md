## 2024-05-24 - Skip Link with Custom Cursors
**Learning:** In environments with custom cursors (where native cursor might be disabled or focus styles overridden globally), native keyboard accessibility mechanisms like "skip to main content" links can easily be forgotten. Because global focus styles are sometimes hidden to match the custom cursor aesthetics, skip links must explicitly maintain strong `focus-visible` styling (using `.focus-visible:ring-2` and clear contrast) so that keyboard-only users can navigate past the persistent header.
**Action:** When adding or verifying global layout wrappers in Next.js applications that hide the default cursor, always check for the presence of a "skip to main content" link as the very first focusable element in the DOM tree. Ensure it has explicit `focus-visible` styling and that its target container (e.g., `#main-content`) is properly defined.
